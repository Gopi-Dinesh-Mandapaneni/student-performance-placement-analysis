**Student Performance \& Placement Analysis**







Project Overview:



I dove into this project to dig through student academic performance and placement outcomes, using exploratory data analysis (EDA) as my toolkit. My main goal? Find out what really moves the needle for exam scores and helps students to get placements. Plus, brush up on some hands-on Python data analysis. The analysis covers data cleaning, visualization, statistics, and business insights and sets the ground for building predictive models.







Problem Statement:



Lots of schools collect mountains of student data, but it's tough to know which factors matter most for grades or placements.







I set out to answer a few questions:



\-> If students study more, do their exam scores go up?



\-> Is attendance a big deal for performance?



\-> What's the impact of internet usage on academic results?



\-> What factors drive placement success?



\-> What traits do the top students share?







Dataset information



The dataset holds 10,000 students' records. and this data has academic, behavioral, and performance-related features.







Features:



\-> study\_hours: Average study hours per day



\-> attendance: attendance percentage



\- sleep\_hours: Average sleep hours per day



\-> internet\_usage: Daily internet usage hours



\-> assignments\_completed: Number of assignments finished



\-> previous\_score: Previous academic score



\-> exam\_score: Current exam score



\-> placement\_status: Placement outcome (Placed / Not Placed)







Technologies Used

Project workflow



1\. Data understanding: checked dataset shape, features, data types, and missing values.



2\. Exploratory data analysis (EDA)



Univariate analysis: Used histograms, KDE plots, and box plots to see how the data spread out, spot skewed numbers, and catch outliers.



&#x20;  - Bivariate Analysis: Fired up scatter plots and correlation checks to dig into how different variables connect.



3\. Placement analysis:



Compared placed versus not placed students, focusing on study hours, attendance, previous scores, exam scores, and assignment completion.



4\. Correlation Analysis



Made a correlation heatmap to spot relationships between numbers and figure out the top predictors for exam scores.



5\. Student Segmentation



Broke students into high and low performers and looked at those who got placed, searching for common traits and behaviors.







Key Insights



Academic Performance



\-> Students who put in more study hours usually see better exam scores.



\-> If you’ve done well before, odds are you’ll do well again.



\-> Completing assignments really supports exam performance.







Attendance



\-> Higher attendance links to better exam scores.



\-> Being consistently present seems to help academically.







Internet Usage



\-> Too much internet use is tied to lower exam scores.



\-> Striking a balance pays off.







Placement Success



Placed students have a few things in common:



\-> Their exam scores are higher



\-> Attendance is better



\-> They’ve finished more assignments



\-> Their academic history is stronger
